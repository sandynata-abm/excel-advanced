Fungsi INDIRECT: The Magic of Text Reference

Mengubah string teks menjadi alamat sel atau range yang valid secara programatik.

Konsep Dasar

Secara standar, jika Anda menulis ="A1", Excel hanya menganggap itu teks biasa. Namun, jika dibungkus =INDIRECT("A1"), Excel akan pergi ke sel A1 dan mengambil nilainya.

REFERENSI LANGSUNG =A1
REFERENSI INDIRECT =INDIRECT("A" & 1)

Kasus 1 Dynamic Sheet Referencing

Bayangkan Anda punya sheet bernama Januari, Februari, dan Maret. Anda ingin mengambil angka Total Penjualan di sel B10 pada masing-masing sheet tanpa menulis rumus satu per satu.

Nama Sheet (Cell A1) Target Cell Rumus INDIRECT Hasil Logika
Januari B10 =INDIRECT(A1 & "!B10") Mengambil data dari Sheet Januari cell B10
Februari B10 =INDIRECT(A1 & "!B10") Mengambil data dari Sheet Februari cell B10

Kasus 2 Named Range Calculation

Misalnya Anda memiliki 2 Named Range: Bonus_Staff dan Bonus_Manager.

// Jika Cell E1 berisi teks: "Bonus_Staff"

// Rumus di bawah ini akan menjumlahkan isi dari range tersebut:

=SUM(INDIRECT(E1))

⚠️ Penting: Volatile Function

Fungsi INDIRECT bersifat Volatile. Artinya, Excel akan menghitung ulang rumus ini setiap kali ada perubahan kecil di sel manapun dalam workbook Anda. Gunakan secukupnya pada file yang sangat besar untuk menjaga performa.
