Tugas 1: Ringkasan Periodik (Grouping)

  1. Buat PivotTable dari data transaksi, lalu taruh field Tanggal ke bagian Rows.
  2. Taruh field Total Penjualan ke bagian Values agar muncul ringkasan angka per tanggal.
  3. Klik salah satu nilai tanggal yang tampil di PivotTable. Pastikan yang diklik adalah tanggal di PivotTable, bukan tanggal di tabel sumber.
  4. Klik kanan pada tanggal tersebut lalu pilih Group.
  5. Pada jendela Grouping, pilih Months dan Quarters. Jika perlu, biarkan Years tetap tercentang atau nonaktifkan sesuai kebutuhan tugas.
  6. Klik OK untuk menampilkan ringkasan periodik berdasarkan bulan dan kuartal.

Jika Group tidak muncul, gunakan PivotTable Analyze > Group Field. Jika tombol itu juga tidak aktif, cek apakah kolom Tanggal masih berupa teks atau ada sel kosong di tengah data.

Tugas 2: Analisis Pertumbuhan (Comparison)

  1. Pastikan PivotTable untuk Tugas 1 sudah menampilkan data per bulan. Jika belum, selesaikan grouping tanggal terlebih dahulu.
  2. Klik salah satu angka di area Values pada PivotTable. Yang diklik harus sel angka, bukan label bulan atau kuartal.
  3. Klik kanan pada angka tersebut lalu pilih Show Values As.
  4. Pilih % Difference From... untuk menampilkan perubahan dibanding periode sebelumnya.
  5. Pada kotak dialog yang muncul, set Base Field ke Tanggal dan Base Item ke (Previous).
  6. Klik OK. Hasilnya akan menampilkan persentase naik/turun penjualan dari bulan sebelumnya.

Jika Show Values As tidak terlihat saat klik kanan, buka Value Field Settings lalu cari tab atau bagian Show Values As. Jika hasilnya tidak berubah, pastikan field nilai yang dipakai adalah Total Penjualan, bukan field lain.

Tugas 3: Dashboard Interaktif (Slicers)

  1. Klik salah satu sel di PivotTable agar tab khusus PivotTable muncul di ribbon.
  2. Buka tab PivotTable Tools lalu pilih Analyze (kadang tampil sebagai PivotTable Analyze).
  3. Klik Insert Slicer.
  4. Centang field Wilayah dan Kategori, lalu klik OK.
  5. Pindahkan slicer ke area yang mudah dilihat, lalu gunakan tombol pada slicer untuk menyaring data secara visual.
  6. Jika ingin mengembalikan tampilan semua data, klik ikon Clear Filter pada slicer.

Jika Insert Slicer tidak terlihat, pastikan yang aktif adalah PivotTable, bukan sheet biasa. Pada Excel 2013/2016, menu ini baru muncul setelah Anda mengklik salah satu sel di PivotTable.

🎨 Professional Layout Design

Ubah desain PivotTable melalui tab Design:

  • Gunakan Report Layout: Tabular Form agar header kolom terlihat jelas.
  • Aktifkan Banded Rows untuk kemudahan membaca.
  • Format angka menjadi Currency (IDR) dengan 0 desimal.
